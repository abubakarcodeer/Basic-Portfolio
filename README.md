# Basic Portfolio

A simple and responsive **personal portfolio website** built using **HTML, CSS, and JavaScript**.  
This portfolio includes multiple sections such as Home, About, Services, Projects, and Contact, along with a Header, Navbar, and Footer.

The Home section features a **typewriter text animation**, and the website uses **smooth scrolling** for better user experience.
